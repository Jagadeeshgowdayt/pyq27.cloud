Question Paper Website - Local Server Setup

Note: This website works perfectly when opened directly in your browser. You only need a local server if you encounter any file loading issues.

Quick Start (No Server Required)

Recommended: Simply open index.html directly in your web browser. All features will work perfectly!

Optional: Local Server Setup

Method 1: Python Server (if Python is installed)

For Python 3:
python -m http.server 8000
For Python 2:
python -m SimpleHTTPServer 8000

Then open: http://localhost:8000

Method 2: Node.js Server (if Node.js is installed)

npx http-server -p 8000

Then open: http://localhost:8000

Method 3: Live Server (VS Code Extension)

  1. Install "Live Server" extension in VS Code
  2. Right-click on index.html
  3. Select "Open with Live Server"
